Java 变量参数的传递

Java 的数据类型可以分为PrimitiveType基本类型和ReferenceType引用类型，当变量作为参数传递给函数的时候，都可以将其看作是值传递。基本类型变量传递的是值本身，引用类型变量传递的是变量引用的值。

1.基本类型变量

对于基本类型变量，看下面的例子

public class ex {
    public static void main(String[] args) {
        int a = 1;
        int b = 2;
        System.out.println("before change");
        System.out.println(a + " " + b);
        change(a,b);
        System.out.println("after change");
        System.out.println(a + " " + b);

    }

    static void change(int a, int b) {
        a = 3;
        b = 4;
    }
}

结果显而易见，两个变量a，b只是将值传递了进去，并不会改变变量本身的值。因此输出为

before change
1 2
after change
1 2

2.引用类型变量

对于引用类型变量，看下面的例子

public class Date {
    int year;
    int month;
    int day;
    public Date(int year, int month, int day) {
        this.year = year;
        this.month = month;
        this.day = day;
    }

    public static void change(Date date) {
        date.year = 2014;
        date.month = 1;
        date.day = 1;
    }

    @Override
    public String toString() {
        String s = "Year: " + year + " Month: " + month + " Day: " + day; 
        return s;
    }

    public static void main(String[] args) {
        Date date1 = new Date(2013, 12, 31);    
        System.out.println("before change");
        System.out.println(date1);
        change(date1);
        System.out.println("after change");
        System.out.println(date1);
    }
}

将变量的引用传递了进去，change函数将得到改变外部date变量的能力，从而在函数内部的更改也会反映到函数外部的变量中去

before change
Year: 2013 Month: 12 Day: 31
after change
Year: 2014 Month: 1 Day: 1

3.String 和基本类型变量的包装类变量

对于.String 和基本类型变量的包装类，看下面的例子

public class finalClass {

    public static void main(String[] args) {
        String s = "abc";
        System.out.println("before change");
        System.out.println(s);
        changeString(s);
        System.out.println("after change");
        System.out.println(s);

        Integer i = 50;
        System.out.println("before change");
        System.out.println(i);
        changeInteger(i);
        System.out.println("after change");
        System.out.println(i);

    }

    public static void changeString(String s) {
        s = "def";
    }

    public static void changeInteger(Integer i) {
        i = 60;
    }

}

得到的结果如下

before change
abc
after change
abc
before change
50
after change
50

既然String和基本类型变量的包装类变量也是引用类型，为什么变量值没有改变呢？因为String和基本类型变量的包装类都是final类型，在函数中对传递进来参数的赋值实际上只是指向了新的对象，并没有改变原来函数外引用的对象，相当于进行了如下操作:

String s1 = "abc";//指向不可变常量"abc"
String s2 = s1;//将引用传递给s2，此时s1和s2都指向"abc"
System.out.println(s1);//"abc"
System.out.println(s2);//"abc"
s2 = "def";//s2引用指向了新的"def"，s1仍然指向"abc"
System.out.println(s1);//"abc"
System.out.println(s2);//"def"
