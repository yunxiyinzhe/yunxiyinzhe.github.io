wordpress 个人博客迁移记---从BAE(Baidu App Engine)迁出 

之前在百度云空间上搞了个wordpress个人博客,一直感觉运行的不错,免费的空间对于一个小型个人博客来说也很充足. 突然前些日子发现BAE从2.0升级到3.0, 需要自己手动迁移. 折腾了很久终于把博客和数据库迁移到3.0平台上了,怎奈之前的域名没法绑定了,只能用新的域名.






索性不再用二级域名了,自己买个域名得了. 于是就去买了这个域名,万网上买的,首年45续费50,价格还算便宜. 没有去godaddy上买是因为近期没有什么优惠. 






域名买好之后解析到在BAE上的地址, 然后BAE上绑定自己的域名,然后又出问题了.之前我记得BAE绑定自己的域名是不需要备案的,貌似现在有要求了,而且备案流程看着我就头疼,还要花很久时间.所以既然我域名都买了我就只能放弃BAE了. checkout最新代码,backup数据库,我去找别的空间去.<!-- more -->






在衡天主机上买的空间,首先纠结于是香港主机还是美国主机,前者访问速度好,后者便宜. 但想想我这个人博客也没什么太好的网站相应需求, 就搞了个美国主机,后来证明速度也还不错的.






主机买好先是登陆不了DirectAdmin面板,联系客服换了服务器IP搞定, 然后域名解析到新的服务器IP, 上传wordpress代码,建立数据库,导入备份数据库,配置wordpress数据库,SQL执行






UPDATE wp_options SET option_value = replace( option_value,'http://dylangao.duapp.com', 'http://dylangao.com';) WHERE option_name ='home' OR option_name 'siteurl' ;
UPDATE wp_posts SET post_content = replace(post_content,'http://dylangao.duapp.com', 'http://dylangao.com'); 
UPDATE wp_posts SET guid = replace( guid,  'http://dylangao.duapp.com', 'http://dylangao.com' ) ;






修改连接,其中经历了各种坎坷,不过好在google和客服的帮助下,顺利完成了迁移. 






自己租借的主机,自己的域名,感觉清爽了很多,也geek了很多.
